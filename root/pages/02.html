POSITION

cztery podstawowe wartości:

  1. static - domyślna, dzięki niej wszystkie elementy są w jednej pozycji i się widzą. dla static top, left i z-index nie dzialają.
  2. fixed dizęki niemu wyjmujemy element ze struktury i przyklejamy do okna przeglądarki. Ustawia pozycję zawsze względem okna przeglądarki. Pododaniu position: fixed element liniowy, np. span staje się elementem blokowym.
  3. absolute - Jest pozycjonowany DOMYŚLNIE względem okna przeglądarki, ale nie jest przyklejony. Żeby był pozycjonowany względem rodzica musi być w nim zagnieżdżony i rodzic musi posiadać position: relative (będzie też działało jeśli rodzic będzie miał position: ustawione na inną wartość)
  4. relative - Pozycjonowanie elementu względem JEGO pozycji, a nie np. okna przeglądarki, czy rodzica. Ciągle jest widziany w układzie, ale możemy go przemieszczać.
